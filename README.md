# Manpreet Singh • Portfolio

A modern developer portfolio built with React, TypeScript, and Vite.

Designed with a clean UI inspired by Duolingo’s playful structure and strong visual hierarchy while keeping the codebase modular, scalable, and maintainable.

This project represents more than a frontend exercise.  
It reflects my long-term focus on software engineering, self-education, and building real systems through consistent practice.

# Design Philosophy

This portfolio focuses on:

- readability over visual chaos
- maintainability over shortcuts
- structure over unnecessary complexity
- strong fundamentals over trend chasing

A surprising amount of modern frontend development seems emotionally committed to making simple things harder than necessary.

This project tries not to contribute to that problem.

# Local Development

Clone the repository:

```bash
git clone <your-repo-url>
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
